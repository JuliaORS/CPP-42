
# CPP

## Project description

This project is a collection of various CPP projects where different concepts are explored.
  - Class and instance
  - List
  - Visibility
  - Pointers to members
  - Filestream
  - Ad-hoc polymorphism
  - Operator overload
  - Canonical form
  - Inheritance C++
  - Sub-typing polymorphism
  - Abstract classes and interfaces
  - Nested classes
  - Upcast and downcast
  - Types of cast
  - Templates
  - Default type
